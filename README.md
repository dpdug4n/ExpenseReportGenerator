# ExpenseReportGenerator
Combines Images of Receipts &amp; PDF of spreadsheet into one PDF

How To Run Script

Install Python - https://www.python.org/downloads/

Download the .bat files and the .py file to the same folder.

Run Installer.bat

Click on the ExpenseReportGenerator.bat file to run the program or 'Right Click ->Send to -> Desktop' to create an icon to run the program.
